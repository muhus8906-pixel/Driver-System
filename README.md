# Driver-System

## Реализованные алгоритмы

### 1. SimpleFinder (Простой поиск)
Линейный поиск с сортировкой. Сложность: O(n log n)

### 2. BucketFinder (Поиск по сегментам)
Разбиение карты на сегменты. Сложность: O(n) в среднем

### 3. GridFinder (Поиск по сетке)
Поиск в квадратной области вокруг заказа. Сложность: O(k), где k << n

## Результаты производительности
